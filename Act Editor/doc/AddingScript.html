There are two ways to use scripts in Act Editor. The following method adds a script permanently and it can be reused. The other method runs a script from a text editor (it can be accessed via Scripts > Script Runner..., more details can be found here.

Adding a permanent script

In Act Editor, open the scripting folder from Scripts > Open scripts folder.

Make a copy of the file script_sample.cs and rename it to whatever you want. The file name only changes the order the script will appear in Act Editor.

If you did everything right, you should now have a new MenuItem in the menu bar with "Custom Scripts". In the Scripts folder, you will also see a new DLL file with the same name as your script (this is handled automatically). This DLL simply avoids recompiling your script everytime.

That's all! The script is ready to use and it reloads automatically if the .cs file is modified.
